# Ignite hands-on

## Build
```sh
./gradlew clean build
```

## Run

### Keynote

```sh
./gradlew runKeynote
```

[http://localhost:8080](http://localhost:8080)


### Server Nodes

```sh
./gradlew runServer1
```

```sh
./gradlew runServer2
```

```sh
./gradlew runServer3
```


### Client Node

```sh
./gradlew runClient
```


### Demo Application

```sh
./gradlew runDemo
```

[http://localhost:8081](http://localhost:8081)
